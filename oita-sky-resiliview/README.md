# OITA SKY RESILI-VIEW

**大分スカイ・レジリビュー** - 「絶景で癒やし、空の目で命を守る。」

## 概要

観光客に大分の美しさを伝えつつ、ドローンによる防災の有効性を啓蒙するフェーズフリーWebVRアプリ。

## 機能

- 🏞️ **観光モード**: 別府湾・湯布院の360度空撮絶景
- ⛑️ **防災モード**: ハザードマップ・避難所表示

## 技術スタック

- [A-Frame](https://aframe.io/) v1.5.0
- Vanilla JavaScript
- CSS3 (グラデーション・アニメーション)

## セットアップ

1. `assets/img/360_view.jpg` に観光用360度画像を配置
2. `assets/img/360_hazard.jpg` に防災用360度画像を配置
3. `index.html` をブラウザで開く

## フォルダ構成

```
├── index.html          # メインアプリ
├── assets/
│   ├── img/            # 360度画像
│   ├── audio/          # BGM・効果音
│   └── icons/          # アイコン
├── css/style.css       # 外部CSS
├── js/app.js           # 外部JS
└── README.md
```
